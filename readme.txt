"sensor.py" : Developper and Device Python3 class for exchange with TD platform
"test.py"   : Test script for "sensor.py"

You must create a file "private.py" in same directory as scripts before using it.

You must include 4 strings in this file:

TD12XX_ID      = <Your modem id here (something like "1F1F")>
TD12XX_KEY     = <Your modem key (something like "0D641FAF")>
DEVEL_LOGIN    = <Your sensor developer account login>
DEVEL_PASSWORD = <Your sensor developer account password>

"private.py" is not publish on public repo.
